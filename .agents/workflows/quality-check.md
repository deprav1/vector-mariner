# ✅ Quality Check — Проверка качества

// turbo-all

> 5 осей проверки перед каждым значимым обновлением.

## Формат вызова

```text
/quality-check
```

## Шаги

1. **Correctness** — все цифры актуальны? источники живые?
2. **Security** — нет XSS в пользовательском вводе? Ссылки не битые?
3. **Code Health** — JS работает? Нет console.error? HTML валидный?
4. **Simplicity** — можно ли упростить без потери информации?
5. **Performance** — страница грузится < 3 сек? Изображения оптимизированы?
