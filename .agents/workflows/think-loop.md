# 🧠 Think Loop — Глубокое обдумывание

// turbo-all

> N циклов рефлексии: каждый цикл критикует предыдущий и улучшает.
> Адаптировано для анализа страховых данных.

## Формат вызова

```text
/think-loop [вопрос] [cycles=3]
```

---

## Шаги

### Цикл 1: Первое решение

1. Собери все имеющиеся данные по вопросу
2. Предложи ответ на основе текущего понимания
3. Запиши assumptions и risks
4. Укажи источники для каждого утверждения

### Цикл 2: Критика

1. Критикуй решение из Цикла 1
2. Проверь: устаревшие данные? однобокие отзывы? маркетинговые обещания vs реальность?
3. Найди слабые места, edge cases, альтернативы
4. Инициируй веб-поиск для перепроверки ключевых утверждений
5. Предложи улучшенное решение

### Цикл 3: Синтез

1. Объедини лучшее из обоих циклов
2. Финальное решение с confidence score
3. Список action items
4. Пометь все неподтверждённые данные как «⚠️ уточнять»

### (Опционально) Цикл N

Повтори критику-синтез для более глубокого анализа.

## Выход

```markdown
## 🧠 Think Loop Result

**Вопрос:** [вопрос]
**Циклов:** N
**Confidence:** [1-10]

### Финальное решение
[решение]

### Key Insights
- [insight 1]
- [insight 2]

### Risks
- [risk 1]

### Sources checked
- [source 1] — дата, статус
```

## Страховая специфика

### Типичные вопросы для Think Loop:
- «Какая страховка лучше для семьи с ребёнком 3+ лет?»
- «Стоит ли переплачивать за Elegant если нет хронических?»
- «Wiener Basic €90 vs Dunav €110 — что реально лучше?»
- «Покроет ли страховка МРТ коленного сустава?»

### Чеклист верификации данных:
1. Цена подтверждена ≥2 источниками?
2. Источник моложе 6 месяцев?
3. Офиц. сайт vs чат — приоритет офиц. сайту
4. Trustpilot рейтинг — сколько отзывов? (3 отзыва ≠ надёжная оценка)
5. Есть ли реальный кейс использования этого покрытия?
