# 📝 Compound Engineering

// turbo-all

> После каждого нетривиального бага или проблемы — запись в docs/solutions/.

## Формат вызова

```text
/compound [описание проблемы]
```

## Шаги

1. Создай файл `docs/solutions/YYYY-MM-DD-тема.md`
2. Заполни:

```markdown
# Problem
[описание]

# Root Cause
[причина]

# Solution
[решение]

# Prevention
[как избежать в будущем]
```

3. Закоммить файл
