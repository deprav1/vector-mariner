# 🏥 Страховка.RS — Сравнение медицинских страховок в Сербии

Открытый калькулятор и гайд по добровольному медицинскому страхованию (ДМС) в Сербии для иностранцев с ВНЖ.

## Что внутри

- 📊 **Таблица сравнения** — 5 компаний × 14 критериев
- 🏢 **Карточки компаний** — пакеты, цены, плюсы/минусы
- 🧮 **Калькулятор подбора** — ответь на 5 вопросов, получи рекомендацию
- ⚠️ **Подводные камни** — хронические, франшиза, период ожидания
- 💬 **Отзывы экспатов** — реальный опыт
- 📋 **Чеклист документов** — интерактивный
