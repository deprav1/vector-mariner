# Duplicate JS Fragment Breaking Site

## Problem
Сайт полностью не работал — таблица сравнения пустая, калькулятор не реагирует, сценарии не кликаются, анимации fade-in не запускаются.

## Root Cause
На строках 588-589 `index.html` находился **дубликат фрагмента** из функции `calculate()`:
```js
}
. ${rec.name}</h4>...`).join('');
}
```
Это вызывало `SyntaxError`, который убивал ВСЮ секцию `<script>` — ни одна функция JS не инициализировалась.

## Solution
Удалены строки 588-589 (мусорный дубликат). Функция `calculate()` корректно заканчивается одной `}` на строке 587.

## Prevention
- При копировании/вставке кода в монолитный HTML — всегда проверять консоль DevTools на SyntaxError
- Использовать `git diff` перед коммитом для ревью изменений
- Рассмотреть вынос JS в отдельный файл `main.js` для лучшей читаемости при дальнейшем росте
