# Wiener Basic: цена €90 → €136/год

## Problem
Цена Wiener Basic на сайте была указана как ~€90/год (из старых чатов и srb.guide).

## Root Cause
Источник srb.guide и Telegram-чаты содержали устаревшую цену. Официальный источник (sindikatnispetrol.rs + wiener.co.rs) показывает:
- New Basic от **€11.39/мес** = **€136.68/год**
- Лимит €1,000–3,000

## Solution
Обновлена цена на «от ~€136/год⁴» в:
- Карточке компании Wiener
- D-объекте JS (таблица сравнения)
- Источник sup-тега изменён на sindikatnispetrol.rs + wiener.co.rs

## Prevention
- Регулярно проверять цены через веб-поиск
- Приоритет: офиц. сайт > агрегаторы > чаты
